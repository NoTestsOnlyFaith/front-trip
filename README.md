## Środowisko lokalne:

### Wymagania:
- Node 22
- Npm

1. Zainstaluj zależności:

```bash
# npm
npm install

```


2. Uruchom serwer lokalny `http://localhost:3000`:

```bash
# npm
npm run dev

```

## Środowisko produkcyjne:

```bash
# npm
npm run build
```

## Lokalny podgląd produkcji:

```bash
# npm
npm run preview
```
